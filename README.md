# NayePankh Foundation — Data Analytics Project

This repository hosts a modular, python-driven analytics pipeline designed to evaluate donor distribution metrics, assess marketing channel conversions, and provide automated strategic reports for the **NayePankh Foundation**.

## 📊 Project Layout Architecture
* `data/`: Contains the base structured relational database (`donor_dataset.csv`).
* `notebooks/`: Playground environment for exploratory data analysis (`donor_analysis.ipynb`).
* `src/`: Main operational codebase run execution script (`nayepankh_analytics.py`).
* `outputs/`: Dynamically rendered performance visual boards and programmatic marketing analysis reports.
